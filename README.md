# File2Base64
Convert a file to Base64 codes, vise versa
將檔案轉換為Base64編碼，反之亦可。

## Library Dependency 需要的庫
* Base64.ahk
* Base64TohBitmap.ahk
* Class_CtlColors.ahk
* ImagePut.ahk
* IsOver.ahk
* OSD.ahk
* StrCompress.ahk
